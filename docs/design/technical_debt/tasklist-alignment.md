# 技术债：任务列表与普通列表混用对齐问题

## 问题

任务列表与普通列表混用时：
- **任务列表**：`☐ ---大间距--- 文本`
- **普通列表**：`--莫名缩进-- 1. 文本`

**根本原因**：OneNote 将普通列表的列表编号（数字/圆点）对齐到任务列表复选框的**右侧**，导致：
1. 任务列表复选框后有巨大空白
2. 普通列表前面有莫名的缩进

## 尝试过的方案

- Tag + List 组合 → 同时显示两个符号，仍不对齐
- 调整 bullet 属性 → 无效或产生其他符号
- 移除额外属性 → 可转换但对齐问题依然存在

**结论**：OneNote 的 Tag 和 List 是两套独立布局系统，无法通过 XML 强制对齐。原生 OneNote 也无法解决此问题。

## 当前状态

✅ 任务列表功能完整（可交互勾选）
⚠️ 与普通列表混用时有对齐差异（OneNote 限制）

## 可能方案

**分块处理**：拆分混合列表为多个块，每块内只有同一类型
- 优点：对齐完美
- 缺点：有序列表编号会重置
